# 📖 DuckGame <a name="about-project"></a>

> 一个高性能的模块化游戏分享平台，采用现代化的设计理念，提供美观的用户界面和流畅的用户体验。

## ✨ 主要特性 <a name="features"></a>

- 🚀 高性能模块化架构
- 💎 现代化UI设计
- 🎨 美观的用户界面
- 📱 响应式设计
- ⚡ 快速加载和渲染

## 💻 快速开始 <a name="getting-started"></a>

### 前置要求

- Node.js
- pnpm

### 安装依赖

```sh
pnpm install
```

### 开发环境运行

```sh
pnpm dev
```

### 构建项目

```sh
pnpm build
```
